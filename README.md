# standard

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run serve

# build for production with minification
npm run build

```

## 注意

	1、分销商ID存为全局变量 vue.prototype.$distributorId

## 配置

	1、config 文件 HTTP 属性为服务器地址
	2、http.js 文件为axios请求二次封装
